Sheet-to-Form Automator Settings

Configure your Google Sheet for automated form filling

Google Sheet Configuration

Enter the URL of your public Google Sheet. Make sure the sheet is publicly viewable.

Sheet Format Requirements

Your Google Sheet must have exactly 4 columns in this order:

  1. FieldName - Descriptive name for the field
  2. Selector - CSS selector or XPath to target the element
  3. SelectorType - Either "css" or "xpath"
  4. Value - The value to enter, or "CLICK" for buttons, "CHECK" for checkboxes

Example Sheet Structure:

FieldName Selector SelectorType Value
Book Title #data-book-title css The Last Horizon
Author Name //input[@name='author'] xpath Alex Rivera
Submit Button .submit-btn css CLICK

Tips:

  • Use browser developer tools to find accurate selectors
  • Test your selectors before adding them to the sheet
  • Use "CLICK" for buttons and "CHECK" for checkboxes
  • The extension supports elements inside iframes
