Configure your Google Sheet for automated form filling

Google Sheet Configuration

Enter the URL of your public Google Sheet. Make sure the sheet is publicly viewable.

Sheet Format Requirements

Your Google Sheet must have exactly 4 columns in this order:

  1. FieldName - Descriptive name for the field
  2. Selector - The value used to find the element
  3. SelectorType - The method to find the element (see supported types below)
  4. Value - The value to enter, or special actions like "CLICK", "CHECK"
